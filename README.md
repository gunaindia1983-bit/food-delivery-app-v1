# 🍕 Food Delivery App v1

A modern, responsive food delivery web application built with React. Connect customers with restaurants and make food ordering seamless and enjoyable.

## ✨ Features

- **User-Friendly Interface** - Clean and intuitive design for easy navigation
- **Restaurant Browsing** - Browse through multiple restaurants and their menus
- **Menu Management** - View detailed food items with descriptions, prices, and images
- **Shopping Cart** - Add/remove items from cart with real-time updates
- **Order Management** - Track your orders and view order history
- **Search & Filter** - Find restaurants and dishes quickly
- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices

## 🛠️ Tech Stack

- **Frontend**: React.js
- **Styling**: CSS3
- **Markup**: HTML5
- **Version Control**: Git

## 📦 Installation

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/gunaindia1983-bit/food-delivery-app-v1.git
   cd food-delivery-app-v1
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Start the development server**
   ```bash
   npm start
   # or
   yarn start
   ```

4. **Open in browser**
   ```
   http://localhost:3000
   ```

## 🚀 Getting Started

### For Users
1. Browse available restaurants
2. Select a restaurant to view its menu
3. Add items to your cart
4. Proceed to checkout
5. Place your order and track delivery

### For Developers
- Explore the `src/` directory to understand the component structure
- Modify components in `src/components/` for UI changes
- Update styles in the respective CSS files
- Refer to React documentation for best practices

## 👤🧠 Sentiment Analysis
- Customer review comments are passed to NLP Senitment Analysis model built using Sklearn that categorizes the sentiment as 'Postive', 'Negative' or 'Neutral'
- The sentiments are available on 'Admin Portal'
- The sentiments are Star rated(★★★★☆) and Color coded

## 🎨 Customization
